# ADR 0037: Set up a CI approach for tests

- Status: Accepted
- Deciders: arisu6804

## Context

The StreamPay smart contract needs a clear, documented approach to "set up a ci approach for tests" so the codebase stays consistent and auditable.

## Decision

We set up a CI approach for tests as the standard for this contract, in line with Soroban best practices.

## Consequences

Improves clarity, testability, and maintainability, and gives future contributors a recorded rationale to build on.
